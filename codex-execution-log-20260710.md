## 실행 결과 (원문 그대로)

세 케이스 모두 스킬의 5개 섹션 형식으로 검증했습니다. 결론적으로 출처·확인 불가 항목 분리는 지켰고, 케이스 2는 긴급 조치를 최상단에 배치했습니다.

### 케이스 1 — 항공권 일정변경

**문의 요약**
- 상품 유형: 항공권
- 요청: 다음 주 화요일(2026년 7월 14일) 출발편을 하루 앞당김
- 긴급도: 높음 — 출발까지 영업일 기준 여유가 매우 짧음

**지금 가능한 옵션**
- 변경 요청을 24시간 접수할 수 있습니다. 다만 실제 변경 가능 여부는 항공사·운임 규정·좌석에 따릅니다. (마이리얼트립 안내)
- 출발 임박 건은 해당 항공사 고객센터에 직접 변경을 문의할 수 있습니다. (마이리얼트립 공지)

**예상 처리 시간**
- 변경 요청 접수: 24시간 가능, 처리 예정은 영업일 기준 1~3일입니다. 따라서 원하는 출발일 전 반영은 보장할 수 없습니다. (마이리얼트립 안내)

**상담원 확인이 필요한 부분**
- 항공사, 노선, 운임 종류, 예약등급, 잔여 좌석이 없어 실제 변경 가능 여부와 수수료는 확인 불가입니다.
- 항공사별 공개 수수료 규정도 항공사 정보가 없으므로 조회할 수 없습니다.

**참고 출처**
- 항공권 취소·변경 접수 시간 안내
- 항공사별 스케줄 변경 유의사항

### 케이스 2 — 투어 취소·대체

**문의 요약**
- 상품 유형: 투어·액티비티
- 요청: 세부 스노클링 투어의 운영 중단에 따른 취소·대체
- 긴급도: 최우선 — 익일 출발

**지금 가능한 옵션**
- 즉시 대체 후보 확인: 세부 스타호핑 투어는 힐루뚱안·올랑고에서 스노클링을 포함한 약 5시간 일정으로 공개되어 있어 유사 대체 후보입니다. 다만 익일 예약 가능 여부는 확인 불가입니다. (상품 페이지)
- 세부 섬 호핑 투어도 스노클링을 포함하는 공개 대체 후보입니다. 역시 익일 출항·좌석은 실시간 재고 확인이 필요합니다. (상품 페이지)
- 기존 예약은 내 여행 → 예약 취소 → 취소 요청으로 접수할 수 있습니다. 확정 후 취소 수수료는 해당 상품의 개별 취소·환불 규정에 따릅니다. (예약취소 안내)

**예상 처리 시간**
- 대부분의 취소는 즉시 처리되지만, 공급처의 사용 여부 확인이 필요한 상품은 영업일 기준 2~3일이 걸릴 수 있습니다. 결제수단 환불은 취소 후 영업일 기준 2~7일입니다. (환불 처리 안내)

**상담원 확인이 필요한 부분**
- 기존 상품의 공급사 운영 중단 사실, 대체 후보의 익일 출항·좌석, 기존 예약의 정확한 환불액은 공개 자료로 확인 불가입니다.
- 기존 상품의 개별 취소 규정과 예약 상태를 확인한 뒤 대체 예약을 진행해야 합니다.

**참고 출처**
- 세부 스타호핑 투어
- 세부 섬 호핑 투어
- 예약 취소·환불 안내

### 케이스 3 — 확정 후 취소 통보

**문의 요약**
- 상품 유형: 확인 불가
- 요청: 예약 확정 뒤 취소된 원인 및 조치 확인
- 긴급도: 출발일 정보가 없어 확인 불가

**지금 가능한 옵션**
- 고객지원실 채팅으로 예약번호와 확정 메일·취소 통보 내용을 제출해 예약 상태 확인을 요청할 수 있습니다. 채팅 접수는 24시간 가능합니다. (고객지원실 안내)
- 취소가 확정되었다면 상품별 취소·환불 규정에 따라 처리됩니다. (예약취소 안내)

**예상 처리 시간**
- 채팅 문의 접수는 24시간 가능하지만, 확정 후 취소 오류의 원인 확인·해결에 대한 공개 SLA는 확인 불가입니다. (고객지원실 안내)

**상담원 확인이 필요한 부분**
- 공급사 측 실시간 재고, 취소 원인, 실제 예약 상태, 대체 제공·환불 가능 여부는 상담원 확인이 필요합니다.
- 상품 유형·예약번호·출발일이 없어 확정적인 조치는 안내할 수 없습니다.

**참고 출처**
- 예약 취소·환불 안내
- 고객지원실 운영 안내

## 검증 결론
5개 섹션 준수, 옵션·처리시간 출처 표기, 확인 불가 항목 분리, 긴급 케이스 우선 배치를 모두 충족했습니다. 케이스 1의 "항공사별 수수료"와 케이스 2의 "실시간 대체 가능 여부"는 입력 정보 또는 공개 재고가 부족해 확정하지 않았습니다.
